pub trait IntoTensorDesc {
    fn into(&self) -> TensorDesc;
}

Describes a conversion into a Tensor Descriptor.

This allows for convenient creation of a new SharedTensor. e.g. (2, 4) -> [2,4] or () -> [] or 2 -> [2]
